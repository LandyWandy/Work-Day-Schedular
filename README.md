# Work-Day-Scheduler
### Technologies Used
- HTML
- CSS
- JavaScript
- JQuery
- Dayjs
- Bootstrap
### Functionality
This project will help you keep track of time and tasks throughout the day by:
#### Displaying
- Grey = Past Time
- Red = Current Time
- Green = Future Time
#### Entry Field
Will save tasks entered to field in local storage
